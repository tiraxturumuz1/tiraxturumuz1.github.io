# Security Policy

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |
| Legacy  | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability within this project (including the backend API, frontend components, or deployment configurations), please report it privately.

### How to report:
1. **Email:** pinetworkcex@gmail.com
2. **Subject:** Security Vulnerability Report: [Brief description]
3. **Details:** Please include a detailed description of the vulnerability, steps to reproduce it, and the potential impact.

### Our Response Process:
- **Acknowledgment:** We will acknowledge receipt of your report within 48 hours.
- **Investigation:** Our team will investigate the reported issue immediately.
- **Fix & Disclosure:** Once a fix is developed and tested, we will release an update and coordinate with you for a responsible disclosure.

Thank you for helping us keep the ecosystem secure!
